2. Set Theory Framework: ZFC + Anti-Foundation Axiom

Why This Matters: The Bootstrap Resolution involves self-referential normative structures where agents validate normative authority through the very capacity that grounds such authority. Standard set theory prohibits such circular membership relations. Aczel's Anti-Foundation Axiom enables mathematically rigorous treatment of self-referential systems without paradox.

Motivation for Non-Well-Founded Sets

Standard ZFC set theory with the Foundation Axiom prohibits circular membership relations. The Bootstrap Resolution requires self-referential normative structures to be mathematically well-defined.

Standard ZFC Axioms:

  1. Extensionality: ∀A∀B(∀x(x ∈ A ↔ x ∈ B) → A = B)
  2. Empty Set: ∃∅∀x(x ∉ ∅)
  3. Pairing: ∀a∀b∃c∀x(x ∈ c ↔ (x = a ∨ x = b))
  4. Union: ∀A∃B∀x(x ∈ B ↔ ∃C(C ∈ A ∧ x ∈ C))
  5. Power Set: ∀A∃B∀x(x ∈ B ↔ x ⊆ A)
  6. Replacement: ∀A(∀x∈A∃!y φ(x,y) → ∃B∀y(y ∈ B ↔ ∃x∈A φ(x,y)))
  7. Infinity: ∃A(∅ ∈ A ∧ ∀x∈A(x ∪ {x} ∈ A))
  8. Choice: ∀A(∀x∈A(x ≠ ∅) → ∃f∀x∈A(f(x) ∈ x))

Aczel's Anti-Foundation Axiom (AFA)

For every directed graph G = (V,E) there exists a unique set assignment that makes each node correspond to a set whose elements are exactly the sets corresponding to its successors.

3. Modal Logic Framework for Capability-Based Propositions

Why Modal Logic: Coordination requires reasoning about what agents can do, not just what they will do. Modal logic provides precise tools for analyzing capability-based coordination without requiring identical belief systems.

Multi-Modal Language

Basic Modal Operators

  • ⟨⟨A⟩⟩φ: Coalition A has joint strategy to ensure φ (ATL)
  • [A stit: φ]: Agent A sees to it that φ (STIT)
  • K_H(A,φ): Agent A knows how to achieve φ (Knowing-How)
  • Can(A,φ,R): Agent A can achieve φ with resources R (LRC)

Temporal Operators

  • Gφ: φ holds globally (always in the future)
  • Fφ: φ holds eventually (sometime in the future)
  • Xφ: φ holds in the next state
  • φUψ: φ until ψ (φ holds until ψ becomes true)

  • S: Set of states

  • d: Action availability function d: Agt × S → 2^Act
  • o: Outcome function o: S × Act^Agt → S
